 
 Enviando Raster para Postgres:

 Parte Power Shell:

	- 1º Acesse o Power Shell no seu computador:
         Digite: "Power shell" em pesquisar

    - 2º No Power Shell, vá até o diretorio onde está localizado (seu arquivo).zip:
         Exemplo: "C:\Users\RibeiroF> cd C:\Temp"

    - 3º Acesse o servidor Vênus:
	     Digite: sftp fribeiro@venus.iocasta.com.br

    - 4º Suba o arquivo zip para o servidor:
         Digite: put(seu arquivo).zip 

    - 5º Acesse o servidor pelo Power Shell:
         Digite: ssh fribeiro@209.126.127.15
	 
    - 6º Veja se foi realizado o upload do arquivo para o servidor:
	     Digite: ls
	  
	- 7º Remova o zip do seu arquivo:cd 
	     Digite: unzip (seu arquivo).zip

	- 8º Envie seu raster já sem o zip para o banco Postgres:
	     Parte 01: raster2pgsql -s 4326 -I -C -M -F -t 100x100 teste.tif geo.teste > raster_import.sql 
		 Parte 02: PGPASSWORD=yhwnaZ69mhpeDri psql -h postgres.iocasta.com.br -p 25432 -U postgres -d dblocation -f raster_import.sql
		  
		 Atenção: Quando importar um segundo raster para uma tabela já populada, remova o -C e insira o "-a"  
		 
		 raster2pgsql -s 4326 -I -M -F -t 30x30 brasil_coverage_2020.tif geo.raster > brasil_coverage_2020.sql
		 PGPASSWORD=yhwnaZ69mhpeDri psql -h postgres.iocasta.com.br -p 25432 -U postgres -d dblocation -f brasil_coverage_2020.sql
		 
		 raster2pgsql -s 4326 -I -M -F -t 30x30 -a brasil_coverage_2021.tif geo.raster > brasil_coverage_2021.sql
		 PGPASSWORD=yhwnaZ69mhpeDri psql -h postgres.iocasta.com.br -p 25432 -U postgres -d dblocation -f brasil_coverage_2021.sql
